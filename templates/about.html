About OptiCrop

OptiCrop is an Artificial Intelligence based Crop Recommendation System. It recommends the most suitable crop by analyzing soil nutrients and environmental conditions.


Objectives

  • Recommend suitable crops.
  • Increase agricultural productivity.
  • Reduce wrong crop selection.
  • Support farmers using AI.

Technology Used

Frontend HTML CSS
Backend Python Flask
Machine Learning Scikit Learn
Dataset Crop Recommendation Dataset
